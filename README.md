# ❄️ Ultimate Glass Rice - Hyprland

A high-end, frosted glass aesthetic for Hyprland v0.54.2+, featuring white accents, snappy animations, and a focus on clean usability.

![Preview](https://github.com/Frqmelikescheese/hypr-glass-dots/blob/master/image(1).png)

## ✨ Features
- **Modern Syntax:** Fully compatible with Hyprland v0.54.2+.
- **Aesthetic:** Frosted glass blur, solid white gradient borders, andsnappy animations.
- **Waybar:** Single long, blurred glass bar with persistent workspaces and system info.
- **Rofi:** Centered, blurred app launcher with a clean search UI.
- **Terminal:** Optimized Kitty config with dark glass transparency.
- **Installer:** One-click automated setup script with ASCII art and animations.

## 🚀 Installation

### 1. Clone the repository
```bash
git clone https://github.com/Frqmelikescheese/hypr-glass-dots.git
cd hypr-glass-dots
```

### 2. Run the Installer
```bash
sudo ./install.sh
```

### 3. Apply Changes
Log out and log back in to enjoy your new Ultimate Glass setup!

## ⌨️ Keybindings
- `SUPER + Q`: Open Terminal (Kitty)
- `SUPER + R`: Open App Launcher (Rofi)
- `SUPER + E`: Open File Manager (Thunar)
- `SUPER + C`: Close Active Window
- `SUPER + M`: Emergency Exit (to login screen)
- `SUPER + [1-9]`: Switch Workspaces
- `SUPER + +/-`: Adjust Brightness (15% steps)
- `SUPER + Print`: Screenshot entire monitor
- `Print`: Select a area to screenshot
- `SUPER + T` Open firefox
- `SUPER + V` Toggle floating for focued window
- `SUPER + H` Open rofi clipboard
## 🛠️ Requirements
- Arch Linux
- A working Hyprland installation
